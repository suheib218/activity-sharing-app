Development Priorities and MVP

The following are the development priorities for the Activity Manager App, categorized and prioritized based on the time and skills required for implementation. The focus is on delivering the Minimum Viable Product (MVP) first to quickly release a functional version of the app. Non-MVP features will be scheduled for later development phases.

Prioritization by Time & Skills

MVP

  • User Authentication: Allow users to create accounts, log in, and log out securely. This is a foundational feature that will be essential for all other user-based functionality.
  • Activity Creation: Users can create new activities by entering the necessary details such as title, description, time, and location. This feature is central to the purpose of the app.
  • Activity Participation: Users can join or leave activities, and the activity creator can see who has joined. This is crucial for building community engagement.
  • Activity Listings: A dynamic list of all activities will be displayed, with filtering and sorting options for users to find the right activity quickly.
  • SQLite Integration: Using SQLite for persistent data storage, ensuring user and activity data is saved and retrievable.

Non-MVP

  • Notifications: Implement email or in-app notifications to inform users when an activity they joined is updated or canceled.
  • Activity Reminders: Add the ability for users to set reminders for upcoming activities they are participating in.
  • Google Maps Integration: Use Google Maps API to show activity locations on a map, allowing users to see where activities are happening in relation to their current location.
  • User Profiles: Enhance user profiles by allowing users to add bio information, photos, and see a history of activities theyâ€™ve participated in or created.
  • Ratings and Reviews: Allow users to rate activities and provide feedback, helping others choose which activities to join.
  • Social Sharing: Enable users to share activities via social media platforms such as Facebook, Twitter, or Instagram to encourage broader participation.
